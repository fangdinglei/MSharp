# 预期支持的C#语法

> 不支持的只是没有测试和特别设计的，可能某些标记为不支持的操作实际上是可用的。
>
> double* 表示使用double能存储的数据类型，如int bool

| 模块         | 功能点         | 示例           | 支持程度                     | 不支持的           |
| ------------ | -------------- | -------------- | ---------------------------- | ------------------ |
| 变量         | 局部变量定义   | double a=1     | double*                      | var 、dynamic      |
|              | 局部变量取值   |                | 完全支持                     |                    |
| 数值         | 字面量         | -1             | double*                      | string、集合等     |
|              | 常量           | UnitConst.xxx  | 游戏常量                     | const定义的变量    |
|              | 变量值         |                | 游戏对象状态、局部变量       | 其他面向对象的求值 |
| 运算         | 基础运算       | a = a + b      | +-*/等                       |                    |
|              | 自增自减       | a++,++a        | 局部变量                     | 非局部变量         |
|              | 小括号         | a = a + (b *c) | 完全支持                     |                    |
|              | 赋值           | a = a + b      | 完全支持                     |                    |
|              | 条件语句       | a>=b           | 不完全支持                   | 运算符短路         |
| 语句或代码块 | if条件         |                | 完全支持                     |                    |
|              | if方法体       |                | 完全支持                     |                    |
|              | for变量初始化  |                | double*                      |                    |
|              | for条件        |                | 完全支持                     |                    |
|              | for方法体      |                | 完全支持                     |                    |
|              | while条件      |                | 完全支持                     |                    |
|              | while方法体    |                | 完全支持                     |                    |
| 函数         | 返回值         |                | double*                      |                    |
|              | 方法体         |                | 基本支持                     | 单行函数定义       |
|              | 普通参数       |                | 完全支持                     |                    |
|              | out等参数      |                | 基本不支持（仅game api支持） |                    |
|              | 默认参数       |                | 不支持                       |                    |
|              |                |                |                              |                    |
|              |                |                |                              |                    |
| 函数调用     |                |                | 内联、堆栈                   |                    |
|              |                |                |                              |                    |
| 类与结构体   | 自定义数据类型 |                | 不支持                       |                    |
|              | this           |                | 不支持                       |                    |
|              |                |                |                              |                    |
|              |                |                |                              |                    |
| 命名空间     |                |                | 不支持                       |                    |
|              |                |                |                              |                    |
